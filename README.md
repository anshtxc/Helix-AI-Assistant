#Helix AI Assistant

Helix AI is a smart voice-controlled desktop assistant that allows users to interact with their computer using natural language commands. It can listen, understand, and perform tasks like opening applications, answering questions, and searching the web.

---

# Features

- 🎤Voice Command Recognition  
- ▶️ Start / Stop Button Control  
- 💻 Open Applications & Perform System Tasks  
- 🌐 Web Search Integration  
- 🤖 AI-Based Responses  
- 🎨 Interactive GUI (Tkinter)  
- 🔊 Text-to-Speech Output  

---

## 🛠️ Tech Stack

- Python  
- Tkinter  
- SpeechRecognition  
- pyttsx3  
- OS & Webbrowser modules
